## ADDED Requirements

### Requirement: 租户管理页面

系统 SHALL 提供租户管理页面，允许平台管理员管理所有租户。

#### Scenario: 访问租户管理页面
- **WHEN** 平台管理员导航到 /tenant-management
- **THEN** 显示租户管理页面
- **AND** 非平台管理员访问时重定向到首页

#### Scenario: 页面布局
- **WHEN** 租户管理页面加载完成
- **THEN** 显示以下区域：
  - 页面标题和描述
  - 统计卡片区（总租户数、正常运营、已冻结、接入车场）
  - 搜索栏和新建按钮
  - 租户列表表格

### Requirement: 租户统计卡片

系统 SHALL 在租户管理页面顶部显示租户统计数据。

#### Scenario: 显示统计数据
- **WHEN** 页面加载时
- **THEN** 调用 API 获取统计数据
- **AND** 显示四个统计卡片：总租户数、正常运营数、已冻结数、接入车场数

#### Scenario: 统计数据实时更新
- **WHEN** 执行租户操作（创建、冻结、解冻）后
- **THEN** 统计卡片数据自动更新

### Requirement: 租户列表

系统 SHALL 以表格形式显示所有租户列表，支持筛选、搜索和分页。

#### Scenario: 显示租户列表
- **WHEN** 页面加载完成
- **THEN** 调用 API 获取租户列表
- **AND** 显示租户信息：公司名称、联系人、车场数、状态、创建时间

#### Scenario: 按状态筛选
- **WHEN** 点击筛选 Tab（全部、正常、冻结）
- **THEN** 列表按对应状态筛选

#### Scenario: 搜索租户
- **WHEN** 在搜索框输入关键词
- **THEN** 列表按公司名称或联系人模糊匹配

#### Scenario: 分页
- **WHEN** 租户数量超过每页数量
- **THEN** 显示分页组件
- **AND** 点击页码切换页面

### Requirement: 新建租户弹窗

系统 SHALL 提供新建租户的弹窗表单。

#### Scenario: 打开新建弹窗
- **WHEN** 点击「新建租户」按钮
- **THEN** 显示新建租户弹窗

#### Scenario: 新建租户表单字段
- **WHEN** 弹窗显示时
- **THEN** 表单包含：公司名称（必填）、联系人（必填）、联系电话（必填）、管理员邮箱（选填）

#### Scenario: 提交新建租户
- **WHEN** 填写表单并点击确认
- **THEN** 调用创建租户 API
- **AND** 成功后关闭弹窗、刷新列表、显示成功提示

#### Scenario: 表单校验
- **WHEN** 提交表单时字段校验失败
- **THEN** 显示对应字段的错误提示

### Requirement: 编辑租户弹窗

系统 SHALL 提供编辑租户的弹窗表单。

#### Scenario: 打开编辑弹窗
- **WHEN** 点击租户行的「编辑」按钮
- **THEN** 显示编辑弹窗，表单预填充该租户的数据

#### Scenario: 提交编辑租户
- **WHEN** 修改表单并点击确认
- **THEN** 调用更新租户 API
- **AND** 成功后关闭弹窗、刷新列表、显示成功提示

### Requirement: 冻结/解冻操作

系统 SHALL 提供租户的冻结和解冻操作，带有二次确认。

#### Scenario: 冻结租户
- **WHEN** 点击租户行的「冻结」按钮
- **THEN** 显示二次确认弹窗
- **AND** 确认后调用冻结 API

- **AND** 成功后更新租户状态、显示成功提示

#### Scenario: 解冻租户
- **WHEN** 点击已冻结租户行的「解冻」按钮
- **THEN** 显示二次确认弹窗
- **AND** 确认后调用解冻 API
- **AND** 成功后更新租户状态、显示成功提示
