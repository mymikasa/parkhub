## ADDED Requirements

### Requirement: 登录页布局

系统 SHALL 实现与设计稿一致的登录页布局。

#### Scenario: 登录页双栏布局
- **WHEN** 访问登录页
- **THEN** 桌面端显示左右分栏布局
- **AND** 左侧为品牌区域 (55% 宽度)，使用渐变背景
- **AND** 右侧为登录表单区域 (45% 宽度)，白色背景

#### Scenario: 品牌区域内容
- **WHEN** 查看登录页品牌区域
- **THEN** 显示以下内容：
  - Logo 和产品名称
  - 主标题 "智慧停车管理平台"
  - 产品描述文案
  - 数据统计卡片（今日通行、接入车场、在线设备）
  - 版权信息

#### Scenario: 登录表单区域
- **WHEN** 查看登录页表单区域
- **THEN** 显示以下内容：
  - 账号登录 / 手机号登录 Tab 切换
  - 账号密码表单
  - 手机验证码表单
  - 第三方登录入口

#### Scenario: 移动端登录页
- **WHEN** 在移动端访问登录页
- **THEN** 隐藏左侧品牌区域
- **AND** 显示顶部 Logo
- **AND** 登录表单占满屏幕

### Requirement: 仪表盘布局

系统 SHALL 实现与设计稿一致的仪表盘布局。

#### Scenario: 侧边栏
- **WHEN** 访问任意仪表盘页面
- **THEN** 显示固定定位的侧边栏
- **AND** 宽度为 256px (w-64)
- **AND** 背景为渐变色 `linear-gradient(180deg, #1e3a5f 0%, #1a3454 100%)`
- **AND** 包含 Logo、导航菜单、用户信息

#### Scenario: 侧边栏导航
- **WHEN** 查看侧边栏导航
- **THEN** 导航按分组组织：
  - 平台管理：租户管理
  - 车场运营：停车场管理、设备管理、计费规则
  - 运营监控：实时监控、出入记录、操作员工作台
- **AND** 当前页面的导航项高亮显示

#### Scenario: 导航项状态
- **WHEN** 导航项为激活状态
- **THEN** 应用 `.active` 样式
- **AND** 背景色 `rgba(37, 99, 235, 0.3)`
- **AND** 左边框 `3px solid #3b82f6`

#### Scenario: 侧边栏用户信息
- **WHEN** 查看侧边栏底部
- **THEN** 显示当前用户头像、姓名、角色
- **AND** 提供退出登录按钮

#### Scenario: 主内容区
- **WHEN** 查看主内容区
- **THEN** 左边距 256px (ml-64)
- **AND** 背景色 #f8fafc

#### Scenario: Header
- **WHEN** 查看页面 Header
- **THEN** 固定在顶部 (sticky top-0)
- **AND** 白色背景，底部边框
- **AND** 显示页面标题和描述
- **AND** 显示状态指示和当前时间

### Requirement: 响应式布局

系统 SHALL 支持响应式布局，适配桌面和移动设备。

#### Scenario: 桌面端布局
- **WHEN** 屏幕宽度 >= 1024px
- **THEN** 显示完整侧边栏
- **AND** 主内容区正常显示

#### Scenario: 移动端布局
- **WHEN** 屏幕宽度 < 1024px
- **THEN** 侧边栏默认隐藏
- **AND** 通过汉堡菜单或手势展开侧边栏
- **AND** 使用 Sheet 组件实现抽屉效果
